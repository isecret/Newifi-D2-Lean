## 说明

本仓库配置来自 [IvanSolis1989/OpenWrt-DIY](https://github.com/IvanSolis1989/OpenWrt-DIY) 并修改

双周周五下午十八点二十分构建最新版本并发布 [release](./releases/latest)

## 插件

- 服务
    - 京东签到服务
    - ShadowSocksR Plus+
    - 上网时间控制
    - 解锁网易云灰色歌曲
    - 动态 DNS
    - UPnp
    - Frp 内网穿透
    - KMS 服务器
- 网络存储
    - USB 打印服务器
    - 硬盘休眠
    - FTP 服务器
    - 网络共享
- VPN
    - ZeroTier
